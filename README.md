# Python Calculator

A simple calculator written in Python.

## Features

- Add
- Subtract
- Multiply
- Divide
- Input validation using `try` and `except`
- Prevents invalid number input

## Concepts Practiced

- Variables
- input()
- int()
- while loop
- if / elif
- try / except
- break

## How to Run

```bash
python 01_calculator.py
```
